# Contributing to Ares.me

Thank you for your interest in contributing! This project teaches developers to build AI agents that matter.

## How to Contribute

### Reporting Issues
- Use GitHub Issues
- Include Python version and OS
- Provide minimal reproduction code

### Adding Sessions
- Follow the existing pattern
- Include working code
- Add clear explanations
- Test your code

### Code Style
- Python 3.11+
- PEP 8 compliant
- Type hints where helpful
- Docstrings for functions

### Pull Request Process
1. Fork the repo
2. Create a feature branch
3. Make your changes
4. Add/update tests
5. Update documentation
6. Submit PR with clear description

## Session Ideas

Want to add a session? Consider:
- What new concept does it teach?
- Is it progressive (builds on previous)?
- Does it have working code?
- Is it explained clearly?
