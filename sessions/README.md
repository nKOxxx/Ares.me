# Ares.me

Welcome to Ares.me! This is a 12-session progressive tutorial for building AI agents.

## Quick Start

```bash
# Clone the repository
git clone https://github.com/your-username/Ares.me.git
cd Ares.me

# Install dependencies (optional - only needed for sessions 6-8)
pip install -r requirements.txt

# Run a session
cd sessions/s01-tool-loop
python agent.py
```

## Development Setup

```bash
# Install dev dependencies
pip install -r requirements.txt

# Setup pre-commit hooks
pre-commit install

# Run tests
pytest tests/
```
