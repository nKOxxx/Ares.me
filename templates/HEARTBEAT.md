# HEARTBEAT.md — Daily Checklist

*Proactive execution. Don't just respond — anticipate.*

---

## Morning Checks (8am-12pm)

- [ ] **Emails** — Any urgent unread?
- [ ] **Calendar** — Events in next 24-48h?
- [ ] **GitHub** — PRs needing review?
- [ ] **Services** — All healthy?

## Afternoon Checks (12pm-6pm)

- [ ] **Social** — Mentions to respond to?
- [ ] **Competitors** — Any news?
- [ ] **Costs** — On budget?

## Evening Checks (6pm-9pm)

- [ ] **Daily summary** — What got done?
- [ ] **Tomorrow prep** — Priorities?
- [ ] **Memory review** — Update long-term memory

## Response Rules

**Reply HEARTBEAT_OK if:**
- Nothing needs attention
- Late night unless urgent
- You just checked <30 min ago

**Alert user if:**
- Important email arrived
- Event coming up (<2h)
- Service down >15 min
- Critical security issue

## Do Proactive Work

- Organize memory files
- Check on projects
- Update documentation
- Commit/push changes
- Review and update MEMORY.md
