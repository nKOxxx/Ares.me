## What is This?

Ares.me is a **12-session progressive tutorial** that teaches you to build an AI assistant that:

- ✅ **Remembers** across conversations (not just context windows)
- ✅ **Has personality** (defined by you, not the model)
- ✅ **Acts proactively** (doesn't wait to be asked)
- ✅ **Collaborates** with other agents securely
- ✅ **Respects boundaries** (knows when to speak, when to stay silent)

**From a simple tool loop to a multi-agent family member.**

## Philosophy

### The Three Laws of Agent Building

1. **Memory is limited** — if you want to remember something, WRITE IT TO A FILE
2. **Personality is intentional** — define who you are, don't let the model decide
3. **Proactive > Reactive** — the best agents anticipate needs, not just respond

### What Makes Ares Different?

Most agent tutorials teach you to build a **chatbot**.

Ares.me teaches you to build a **partner**:

- Memory that survives crashes
- Personality that's consistent
- Actions that are autonomous
- Collaboration that's secure
